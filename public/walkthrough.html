Start up

This Walkthrough is done through SET version 7.7.5 installed on an Ubuntu System

Go to the SET folder on your system

$ sudo python setoolkit

read the T&C's and confirm if you agree you will be presented with the menu option Fig 1.1 press 4 to update the toolkit to the lastest version . Type 1 for a list of social enginnering attacks or type 2 to go to a short selection of penetration testing. if you changed values inside the config files press 5 to update to the new configuration

About each option

1) Spear-phishing

Spear-phishing is phishing in a targeting manner. Attackers send fraudulent emails to gather confidential information.This option allows you send out fraudulent emails.

2) Website attack vectors

This module allows you to utilize multiple web-based attack on an intended victim , such as a Credential Harvester or Metasploit browsers exploit.

3) Infectous media generator

Create Infectous payloads to be put on a USB/CD/DVD

4) Create a payload and Listener

Create malware to be used in other modules

5) Mass Mailer

Send out mass emails to multiple emails

6) Arduino-based Attacks

Requires additional hardware to use this module

7) Wireless Access Point Attaks

Send out mass emails to multiple emails

8) QRCode Generator Attaks

Send out mass emails to multiple emails

9) Powershell Attaks

Send out mass emails to multiple emails
