Start up

This Walkthrough is done through SET version 7.7.5 installed on an Ubuntu System

Go to the SET folder on your system

$ sudo python setoolkit

read the T&C's and confirm if you agree you will be presented with the menu option Fig 1.1 press 4 to update the toolkit to the lastest version . Type 1 for a list of social enginnering attacks or type 2 to go to a short selection of penetration testing. if you changed values inside the config files press 5 to update to the new configuration

1) Spear-phishing

Spear-phishing is phishing in a targeting manner. Attackers send fraudulent emails to gather confidential information.This option allows you send out fraudulent emails.

2) Website attack vectors

This module allows you to utilize multiple web-based attack on an intended victim , such as a Credential Harvester or Metasploit browsers exploit.

3) Infectous media generator

Create Infectous payloads to be put on a USB/CD/DVD

4) Create a payload and Listener

Spear-Phishing

Quick notes

Selecting Spear-Phishing will allow you to create a payload to attach to pdf or other file to then be sent through email. Gmail and Yahoo tend to block all pdf's with exe embedded inside of them. Using hotmail allows more of a chance of your attack not being blocked before its even sent. For best results if you have your own emailing server use that.

Steps

To begin Select option 1 (if you want to create a payload/template for use again select the other options).
Next select a payload you would like to use (fig 3.2)
Enter an IP address for the reverse , then select what kind of payload you want (fig 3.4)
depending on what payload you selected continue to flow through the menu's selecting each option you would like Metasploit will have to generate the payload for you and this may take sometime so allow for this.
After going through your steps you will then choose between emailing 1 person or multiples (fig 3.8) You can then use a pre defined template or create your own tailered email.

Website attacks

Quick notes

There are quite a few options for web attacks.
Java Applet spoofs a java certificcate to deliever a metasploit payload. However there could be problems with newer versons of java that dont allow self signing
Metasploit Browser Exploit uses exploits in iframe to deliever Payloads
Credential Harvester allows you to read login/password info from a cloned webpage
TabNabing waits for the user to swap tabs then loads something different in the mean time
Web Jacking makes highlighted links look legitmate but when clicked go to a malicous link
Multi attack ustilise multiple web attack options together
HTA allows for powershell injection through HTA files

Let Set and metasploit prep everything for you and wait for your site to visited and your java to run (fig 4.8) example of facebook login(fig 4.9)

Infectous media generator

Quick notes

Have a usb/cd to export the payload to .usb is easier and if you where to leave a usb lying background someone is likely to pick it up and use it and this can be you way into their system.

steps

To be choose an option (fig 5.1) if you chose option 1 enter in the reverse ip ie you ip address then select a payload (fig 5.2) After selecting a payload select what you want your payload to do. (fig 5.3) Eneter in your ip address and the Listener port. Wait for the payload to be created follow the link provided to get your payload (fig 5.4) and export to a USB. You can then launch the Listener and leave the usb at a desk and see if anyone uses it.
if you get a hit like shown in (fig 5.5) you can run $ sessions -i to bring up active connections . Running $ sessions -i (id number of active connection) to then begin interacting with said connection (fig 5.6) running $ sysinfo to bring up details about the connection.$ help brings up options yu can use while connected.

Wireless Access

Prerequisite

$ apt install aircrack-ng
$ apt install dsniff
$ apt-get install isc-dhcp-server

Quick notes

If you are using a Virtual machine you may not be able to acces your wireless device. You will need to have a usb wifi router to spoof the wireless access point.

Steps

A simple set up just select option1 (fig 8.1) to Start
pres enter to procede to the config file just press (ctrl + x ) to leave this and continue (fig 8.2).
Choose an ip address range you want to give (fig 8.3) Choose you wifi routers name should be wlan0 but to check open a new termial and Type $ ifconfig -a it will give a list of all network adapters.
After you have typed in your equilivalent to wlan0 set will start the wifi up for you .
